void TIM1_UP_TIM16_IRQHandler ( void  )

This function handles TIM1 update interrupt and TIM16 global interrupt.

Note
This function start the acquisition of the feedback current from the power module.
The acquisition is synchronous with the PWM to prevent error from the switching noise of the transistor.

◆ TIM7_DAC_IRQHandler()

void TIM7_DAC_IRQHandler ( void  )

This function handles TIM7 global interrupt, DAC2 and DAC4 channel underrun error interrupts.

Note
This function finishes the startup sequence.
This function also reset the speed such as to prevent the power module from encountering an error.
